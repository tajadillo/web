/*========== Transparencia ==========*/
    En el modelo HSL (Hue, Saturation, Lightness) no se incluye la transparencia directamente, ya que esta se maneja en el modelo HSLA (Hue, Saturation, Lightness, Alpha). El canal alfa (A) define el nivel de transparencia de un color y se representa como un valor entre 0 (completamente transparente) y 1 (completamente opaco).
--contrast-accent-color: hsl(190, 70%, 40%); /* Turquoise Blue (Contrasting Accent) */
Para lograr un color transparente utilizando HSLA, puedes usar este formato:

css
--transparent-color: hsla(0, 0, 0, 0.5);
color: hsla(240, 100%, 50%, 0.5);

Aquí:
240 es el tono (Hue), que representa el color.
100% es la saturación (Saturation), que indica qué tan vivo es el color.
50% es la luminosidad (Lightness), que define qué tan claro u oscuro es el color.
0.5 es el nivel de opacidad (Alpha), donde 0 es completamente transparente y 1 es completamente opaco.
Si necesitas transparencia en un color HSL, simplemente agrégale el componente alfa utilizando hsla en lugar de hsl.
