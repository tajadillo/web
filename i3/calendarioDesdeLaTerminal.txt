El comando para ver el calendario desde la terminal de Linux es:

bash
Copiar código
cal
Detalles y Opciones:
Ver el calendario del mes actual:

bash
Copiar código
cal
Muestra el mes actual con el día actual resaltado.

Ver el calendario de un año específico:

bash
Copiar código
cal 2024
Muestra todos los meses del año 2024.

Ver un mes específico de un año:

bash
Copiar código
cal 11 2024
Muestra el mes de noviembre del año 2024.

Mostrar el día juliano (número del día en el año):

bash
Copiar código
cal -j
Muestra el calendario del mes actual con números de día juliano.

Iniciar la semana en lunes:

bash
Copiar código
cal -m
Cambia el inicio de la semana a lunes.

Calendario perpetuo (calendarios históricos o futuros):

bash
Copiar código
ncal -p
Muestra cómo cambiar el calendario de formato gregoriano a juliano o viceversa según la época.

Alternativa con ncal (opcional):
El comando ncal proporciona una vista más detallada y personalizada del calendario:

Mostrar el calendario en formato vertical:
bash
Copiar código
ncal
Mostrar el calendario del mes actual resaltando la semana actual:
bash
Copiar código
ncal -w
Ambos comandos (cal y ncal) están disponibles en la mayoría de las distribuciones de Linux y son muy útiles para consultas rápidas.

No necesariamente. Tanto cal como ncal suelen venir preinstalados en la mayoría de las distribuciones de Linux, ya que forman parte del paquete básico util-linux, que incluye varias herramientas esenciales.

Verificar si están instalados
Para comprobar si cal o ncal están disponibles en tu sistema, puedes escribir en la terminal:

bash
Copiar código
cal
o

bash
Copiar código
ncal
Si obtienes un mensaje como "command not found", entonces no están instalados.

Instalar cal o ncal
Si no los tienes instalados, puedes hacerlo fácilmente. Depende de la distribución de Linux que uses:

En distribuciones basadas en Debian/Ubuntu:

bash
Copiar código
sudo apt install bsdmainutils
En distribuciones basadas en Red Hat/Fedora:

bash
Copiar código
sudo dnf install util-linux
En Arch Linux:

bash
Copiar código
sudo pacman -S util-linux
En openSUSE:

bash
Copiar código
sudo zypper install util-linux
Después de instalarlo, prueba nuevamente con el comando cal o ncal. ¡Deberían funcionar sin problemas!
